# 写在最前

1. **三点密码**是一种字符替换式密码
2. 程序开始后，如果没有密钥文件，将自动生成密钥文件
3. 将密钥文件**发送给另一方**，并保存在**编译器目录**，即可保证双方加解密算法相同
4. 若不需要二次加密，请前往仓库中复制[全0文件](/全0文件.txt)，替换txt内容
5. **直接输入**需编码或解码内容即可
6. 支持的范围：英文大小写字母，**英文**（**半角**）符号，空格
7. 支持的英文（半角）符号如下：

````
 , ! @ # $ % ^ & *

 ( ) < > . ? / ; :

" [ ] { } ' + - ~
````
